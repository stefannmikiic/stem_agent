# STEM Agent Frontend

React + Vite dashboard for the self-evolving QA loop.

## Run

```bash
cd frontend
npm install
npm run dev
```

## What it does

- Shows the current strategy, pipeline stages, recent failures, and learned rules.
- Lets you import `error_memory.json` or `learned_rules.json` from the Python app.
- Exports the current dashboard state as a single JSON snapshot.
